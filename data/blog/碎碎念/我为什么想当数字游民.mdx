Define Fear
1. 什么事情是你因为害怕没有去做的
2. 不做你会错过什么
3. 你为什么不去做那些事情
4. 最坏的结果是什么
5. 如果遇到了最坏的结果，你应该怎么补救
6. 如果回到一开始选择的时候，你会怎么选择

Define Goal
1. List 5 things you want
    * To have
    * To Do
    * To Be
2. From there is 15,Choose the top 4
3. Come up with 3 Action items for each
4. Do the first action,right now
